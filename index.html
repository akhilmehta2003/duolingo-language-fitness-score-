Measuring Real Proficiency

While streak counts and XP measure daily engagement, they fail to answer a learner's most critical question: "How fluent am I?"

The Language Fitness Score (LFS) aggregates performance metrics across all cognitive language dimensions to output a dynamic, CEFR-aligned index from 0 to 100.

THE CHALLENGE

Streak Anxiety vs. Real Fluency

"I have a 450-day streak in Spanish, but I froze when trying to order a coffee in Madrid." — Duolingo User Interview

The Core Problem: Gamification mechanics (streaks, hearts, XP) can create hollow habits. Users optimize for keeping the streak alive rather than building communicative confidence.

The Insight: Users have an intrinsic desire for competence. Transitioning users from extrinsic rewards (streaks) to intrinsic progress indicators (proficiency) increases long-term retention and satisfaction.

THE PROPOSED SOLUTION

Language Fitness Score

A multi-dimensional fluency dashboard directly integrated into the core Duolingo journey:

  • Dynamic Scoring: Weighted scores across 5 sub-skills: Reading, Listening, Speaking, Writing, and Vocabulary.
  • Skill Gap Analysis: Translates performance data into actionable "opportunities" (e.g. Pronunciation Accuracy: 68%).
  • AI-Driven Roadmap: Automatically structures a 4-week practice schedule based on the user's weaknesses.
  • Real-Life Readiness: Translates abstract percentage metrics into situational confidence indicators (e.g., Hotel Check-in: 70% ready).
